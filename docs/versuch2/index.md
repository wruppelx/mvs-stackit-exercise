# Versuch 2  
## Content-Auslieferung über CDN

In diesem Versuch wird die Auslieferung transcodierter Videoinhalte über ein Content Delivery Network (CDN) untersucht.

Ziel ist es, die Performance, Skalierbarkeit und Funktionsweise einer verteilten Content-Auslieferungsarchitektur zu analysieren.

---

## Überblick

Im Rahmen dieses Versuchs werden folgende Aspekte behandelt:

1. Grundlagen von Content Delivery Networks  
2. Konfiguration von Fastly als CDN  
3. Anbindung des STACKIT Object Storage als Origin  
4. Analyse der Cache-Mechanismen  
5. Performance-Messungen  
6. Bewertung der Ergebnisse

## Ziel des Versuchs

Ziel ist es, ein Verständnis zu entwickeln für:

- die Funktionsweise verteilter CDN-Infrastrukturen  
- Caching-Strategien und TTL-Konzepte  
- Performance-Optimierung bei Medienauslieferung  
- die Rolle von Origin-Servern im CDN-Kontext
