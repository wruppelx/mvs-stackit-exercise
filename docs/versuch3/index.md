# Versuch 3  
## Automatisierter VoD-Workflow

Dieser Versuch behandelt den Aufbau eines automatisierten
Video-on-Demand-Workflows in der Cloud.  
Im Fokus stehen die Integration der einzelnen Komponenten,
deren Zusammenspiel sowie die Bewertung der Gesamtarchitektur.

---

## Überblick

2. Analyse des Ingest-Prozesses  
3. CDN im Realbetrieb  
4. Umsetzung eines Automatisierungsbeispiels  
5. Fazit und Bewertung

## Ziel des Versuchs

Ziel dieses Versuchs ist es, einen vollständigen und reproduzierbaren
VoD-Workflow aufzubauen, der ohne manuelle Eingriffe funktioniert.
Dabei wird untersucht, wie:

- Upload-Prozesse automatisiert ausgelöst werden können  
- Transcodierung automatisiert gestartet wird  
- Ergebnisse automatisch im Object Storage bereitgestellt werden  
- Inhalte über ein CDN performant ausgeliefert werden  

Der Fokus liegt auf der technischen Durchgängigkeit
vom Upload bis zur Auslieferung.
